Overview

Two Powerful Paradigms, One Agent

Fraktál merges two elite AI coding architectures into a single, cohesive framework — delivering high-performance reasoning at low cost.

Fable 5 Reasoning

Elite reasoning discipline — hypothesis-driven thinking, evidence-based claims, adversarial self-verification, multi-provider LLM orchestration, persistent memory, and MCP server for Reasonix interop.

Grok 4.5 Build Pipeline

Battle-tested multi-agent workflow — plan → design → implement → review. Agent, persona, role, and skill architecture proven in Grok Build, now powered by DeepSeek's API.

Specialized Outputs

Seven domain playbooks guide every build — dashboards, reports, websites, datasets, research, presentations, and infographics. Every output meets H Heuristics professional standards.

High Performance, Low Cost

DeepSeek-first architecture — 10–20× cheaper than frontier models for equivalent quality. Fast inference, 128K context, and chain-of-thought reasoning via DeepSeek R1.

Build Pipeline

Plan → Design → Implement → Review

The flagship Grok Build workflow — four phases, each handled by specialized sub-agents with DeepSeek API powering every step.

1

Plan

Architect explores the codebase and produces a step-by-step implementation plan.

→
2

Design

Writer produces DESIGN.md. Reviewer audits it. Writer revises until clean.

→
3

Implement

Coder writes code matching existing patterns. Verifier checks. Loop until clean.

→
4

Review

Final review against the diff. Security audit on high effort. Gate before shipping.

Capabilities

Everything You Need to Ship

Fraktál packs 9 agent roles, 8 personas, 5 pipeline skills, 7 domain playbooks, and 8 tools into one Python package.

🧠

Multi-Agent Orchestration

Architect, Coder, Verifier, and Reporter sub-agents — each with specialized system prompts and tools. The Orchestrator delegates and synthesizes.

💾

Persistent Memory

SQLite-backed memory survives across sessions. The Orchestrator recalls past decisions and task outcomes automatically.

🔌

MCP Server

Expose all Fraktál tools to Reasonix and any MCP-compatible AI agent via stdio. Drop-in interop with your existing workflow.

📋

Domain Playbooks

Seven playbooks guide specialized outputs — dashboards, reports, websites, datasets, research, presentations, and infographics.

🔍

8 Built-in Tools

Read, write, list files. Execute shell commands. Regex code search. Plus H Heuristics domain tools — dashboard validation, data sources, market sizing.

🎯

DeepSeek-First, Multi-Provider

Default: DeepSeek V3 + R1. Also supports OpenAI, Anthropic, Ollama, and LM Studio via the provider registry.

Quick Start

Up and Running in Under a Minute

Three steps from zero to your first Fraktál build.

1

Install & Configure

Clone, install with pip install -e ., and set your DEEPSEEK_API_KEY environment variable. Run fraktal setup to initialize.

2

Verify Connection

Run fraktal health to confirm API connectivity. Check available models with fraktal models. Browse agents, personas, and skills.

3

Run Your First Build

Execute fraktal build "your task" for the full pipeline — plan, design, implement, review. Or use individual commands for specific phases.
